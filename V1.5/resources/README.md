# Resources Directory

This directory contains application resources including:

- **icons/**: Application and plugin icons
- **styles/**: CSS/QSS style sheets
- **images/**: Images and logos
- **fonts/**: Custom fonts (if any)

## File Organization

- Place the Root3Power logo (`root3_logo.png`) in this directory
- Plugin-specific resources should be in subdirectories named after the plugin
- Use consistent naming conventions for icons (e.g., `action_name_16.png` for 16x16 icons)

## Supported Formats

- **Images**: PNG, JPG, SVG
- **Icons**: PNG (16x16, 24x24, 32x32, 48x48)
- **Styles**: QSS (Qt Style Sheets)
